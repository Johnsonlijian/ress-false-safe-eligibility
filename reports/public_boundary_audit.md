# Public boundary audit

## Included

- Derived CSV tables from existing scored predictions and existing derived PGA summaries.
- Exported SVG/PDF/PNG figures.
- Lightweight scripts that operate only on packaged derived CSV files.
- Source links for upstream third-party datasets, reports, and standards.

## Excluded

- Active manuscript files in Markdown, DOCX, or PDF form.
- Cover letters, reviewer responses, gate reports, decision logs, and internal round folders.
- Raw strong-motion records, downloaded archives, OpenSees output databases, and model checkpoints.
- Credentials, local machine paths, and user-profile paths.

## Current status

The package is local and GitHub-ready after the cleanliness script passes. GitHub repository creation and push remain human-approved steps.
